IPython brings interactive computing via notebooks to let you author and execute code iteratively. On the Google Cloud, notebooks let you work directly with cloud services and APIs such as BigQuery and Google Cloud Storage.

Notebooks let you combine SQL, python scripts, markdown content and charts to query, transform, analyze and visualize your data. Notebooks can be shared with other developers in your cloud project, as well as used to share and publish your results.

Not sure where to start? We've pre-loaded a few sample notebooks to help you get going that walk you through the basics from the notebook metaphor to working with BigQuery and various python libraries.
