Use Cloud DataLab to gain insight from your data. Interactively explore, transform, analyze and visualize your data using Google BigQuery, Google Cloud Storage and python.

Cloud DataLab builds on top of IPython. IPython Notebooks let you combine SQL, python scripts, markdown content and charts in a simple web interface. Notebooks can be shared with other developers in your cloud project, as well as used to share and publish your results.

Not sure where to start? We've pre-loaded a few sample notebooks to help you get going that walk you through the basics from the notebook metaphor to working with BigQuery and various python libraries.
